🧪 Cách test chức năng:

  1. Truy cập Project Admin: http://localhost:3001/project
  2. Chuyển sang tab "Categories"
  3. Bật Edit Mode
  4. Thêm category mới:
    • Click "Add Category"
    • Điền thông tin: Category ID, Title, Project Count, Navigation Path, Display Order
    • ✅ Upload hình ảnh: Drag & drop hoặc click để chọn file
    • Click "Create Category"
  5. Edit category hiện có:
    • Click vào category card để edit
    • Thay đổi thông tin nếu cần
    • ✅ Thay đổi hình ảnh: Click icon hình ảnh để thay thế hoặc X để xóa
    • Click "Update Category"
  6. Verify kết quả:
    • Hình ảnh hiển thị đúng trong category card
    • Hình ảnh được lưu vào database dưới dạng base64
    • Website chính hiển thị hình ảnh đúng
